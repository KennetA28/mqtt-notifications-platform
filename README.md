# Recommended

This project works with bun 1.1.21 Windows version
If you don't have bun installed yet, you can download it here: <a href="https://bun.sh/docs/installation">Bun docs</a>

You can run this project with Nodejs If you prefer

## Enviroments

For set env you can include the next argument in command:
```CMD
 your_start_command && node --env-file=.env
```
This argument set env file, it's not necesary you install dotenv package. You must have Bun, Nodejs 22 or newer
